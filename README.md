# Medical Data Processing Demo

一个简单的医学数据处理示例项目，展示 Python 在医学信号处理中的应用。

## 功能

- ✅ ECG（心电图）信号模拟与处理
- ✅ 滤波（带通滤波、去噪）
- ✅ 特征提取（R波检测）
- ✅ 可视化展示

## 技术栈

- Python 3.8+
- NumPy
- SciPy
- Matplotlib

## 快速开始

```bash
# 安装依赖
pip install numpy scipy matplotlib

# 运行 demo
python ecg_processing.py
```

## 示例输出

运行后会生成：
1. 原始 ECG 信号图
2. 滤波后的信号图
3. R 波检测结果
